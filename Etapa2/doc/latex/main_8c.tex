{\footnotesize\ttfamily int main (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Função principal do programa. 

Esta função é responsável por coordenar a execução principal do programa, que envolve\+:
\begin{DoxyItemize}
\item Carregar o grafo de antenas a partir de um ficheiro de texto
\item Calcular as adjacências entre antenas
\item Mostrar o grafo e as suas adjacências
\item Gravar o grafo num ficheiro binário
\item Ler o grafo a partir do ficheiro binário
\item Limpar os marcadores de \char`\"{}visitado\char`\"{} antes de uma procura em profundidade
\item Executar uma procura em profundidade no grafo
\item Calcular todos os caminhos possíveis entre duas antenas especificadas pelo utilizador
\end{DoxyItemize}

\begin{DoxyReturn}{Returns}
int Retorna 0 no fim da execução 
\end{DoxyReturn}
